# Project Task Plan

## Epics and User Stories

- **Epic: User Management**
  - As a new user, I want to register an account so that I can access the website.
  - As a user, I want to log in so that I can view my tasks.

- **Epic: Task Management**
  - As a user, I want to add a daily task so that I can organize my day.
  - As a user, I want to mark a task as done so that I can track completion.
  - As a user, I want to view my daily tasks so that I can see what I need to do.

- **Epic: Progress Tracking**
  - As a user, I want to see a graph of task completion over time so that I can visualize my progress.
  - As a user, I want to see progress metrics so that I can understand my productivity.

- **Epic: Ranking System**
  - As a user, I want my rank to be calculated based on daily progress so that I can see my standing.
  - As a user, I want to view my current rank so that I can compare with others.

- **Epic: Hosting**
  - As a developer, I want the website hosted for free so that users can access it.
